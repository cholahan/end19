## Quick Start

1. **Install Jekyll**

   ```sh
   gem install bundler jekyll
   ```

2. **Install gem**

   ```sh
   bundle install
   ```

3. **Start the development server**

   ```sh
   bundle exec jekyll serve
   ```
   or

   ```sh
   jekyll serve
   ```
